Installation
============

Requirements
------------

- PHP 8.4 or newer
- `ext-bcmath`

Install with Composer
---------------------

.. code-block:: bash

   composer require infocyph/game-draw

Import
------

.. code-block:: php

   use Infocyph\Draw\Draw;

   $draw = new Draw();
